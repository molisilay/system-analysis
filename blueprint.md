# 🗺️ 智慧維保平台 v2.0｜跨部門協作藍圖
（每個部門：找誰 × 要什麼 × 給什麼）

## 🏛️ IT 資訊部
- 🎯 共同任務
  - 三大字典資料庫落地（設備 / 備品 / 故障代碼）
  - 系統間 API 介接（ERP / MES / WMS / HR）
  - 帳號權限 SSO 整合
  - 資安政策與雲端架構（GCP）
  - 備援 / 災復 / 監控
- 📤 IT 要產出
  - 資料庫 ER 圖 + API 規格書
  - SSO 接入方案 + Token 規則
  - 資安檢查清單
  - 雲端費用月報
- 📥 提升 IT 部門價值
  - 系統使用率 / 異常告警
  - 統一資料源減少維運痛點
  - 數位轉型 KPI 績效

## 👥 HR / 環安人資
- 🎯 共同任務
  - 員工主檔同步（誰能登入）
  - S-6 領料 / 簽核權限規則
  - 班別 / 排班資料介接
  - 教育訓練綁定系統使用率
  - 高風險設備清單 + 事故通報整合
  - 個人防護裝備 (PPE) 與工單關聯
- 📤 HR 要產出
  - 員工編號 / 職位 / 部門 異動 webhook
  - S-6 表單規則 + 簽核層級
  - 班別表 / 假日行事曆
  - 高風險設備分級
  - PPE 規範與穿戴清單
- 📥 提升 HR 部門價值
  - 培訓參與度 / 系統使用報表
  - 工安事故自動通報 / 統計
  - 違規簽核留痕證據

## 🛒 採購部
- 🎯 共同任務
  - 備品料號統一編碼（M5 / M6 基礎）
  - 供應商主檔 + 績效資料
  - 請購 → 採購 → 收料流程上線
  - 委外維修發包流程
  - 交期預測與安全庫存連動
- 📤 採購要產出
  - 料號對照表（舊 vs 新編碼）
  - 供應商分級規則 + 黑白名單
  - 採購 SLA / 歷史交期
  - 委外維修廠商清單
- 📥 提升採購部門價值
  - 自動補貨建議
  - 供應商交期準確度報表
  - 委外維修品質 / 成本對比

## 📦 倉管 / 物管

  - 領料 / 還料流程數位化

- 📤 倉管要產出
  - 儲位主檔 / 倉別代碼
  - 庫存基期 (盤點起始值)
  - 領料 SOP
- 📥 提升倉管部門價值

## 📋 PM 使用方式
- 三步驟協作法
  - 1️⃣ 找誰（看任務性質）
  - 2️⃣ 要什麼（對方先給的資料）
  - 3️⃣ 給什麼（提升對方部門價值的回饋）
- 三輪開會節奏
  - 第一輪 盤點（資料 / 現場）
    - IT / 設備 / HR / 採購 / 倉管
    - 🚀 專案拓展（先談廠區 / 設備位置 / IoT 佈線）
  - 第二輪 規則（政策 / 標準）
    - 財務 / 法務
    - 🚀 專案拓展（再談 SLA / 跨廠節奏 / ISO）
  - 第三輪 決策
    - 高層 Steering Committee
  - 💡 拓展部要分兩次找
    - 第一次（盤點期）：哪幾條產線、設備擺哪、可協調的停機時段
    - 第二次（規則期）：客戶 SLA 對設備優先級、跨廠擴展先做哪間
